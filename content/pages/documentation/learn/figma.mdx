### Accordion

#### Resizing behavior after editing

<DBNotification variant="standalone" semantic="informational">
Known Figma bug: changing the item count or item state (open/collapsed) can reset the resizing behavior of accordion items. Re-apply "Fill" or "Hug" after editing.
</DBNotification>

### Button
#### Multi-line text within Full Width Buttons

<DBNotification variant="standalone" semantic="informational">
Only use multi-line text in exceptional cases, such as when the text is extended due to translation.
</DBNotification>


To enable automatic line breaks in a full-width button, select the "Text" layer inside the button and set the width resizing behavior from "Hug" to "Fill".

### ControlPanel

#### Add a custom logo with the DB Logo Importer

<DBNotification variant="standalone" semantic="informational">

Create your brand-compliant custom logo SVG via the marketing portal. The plugin import ensures system-conformant color variables and sizing.

</DBNotification>

In the Brand area of the ControlPanel, you can add a brand-compliant custom logo. Use the Figma plugin <DBLink href="https://www.figma.com/community/plugin/1590462040996560100/concept-db-logo-importer" target="_blank" rel="noopener noreferrer" variant="inline">DB Logo Importer</DBLink> to import an SVG from the <DBLink href="https://marketingportal.extranet.deutschebahn.com/marketingportal/Marke-und-Design/Basiselemente/Logo/Logozusatz-mit-Tool#unit=Regio&mode=unit&breaking=true" target="_blank" rel="noopener noreferrer" variant="inline">DB brand logo generator</DBLink>.
