### Accordion

#### Resizing behavior after editing

<DBNotification variant="standalone" semantic="informational">
Known Figma bug: changing the item count or item state (open/collapsed) can reset the resizing behavior of accordion items. Re-apply "Fill" or "Hug" after editing.
</DBNotification>

### Button
#### Multi-line text within Full Width Buttons

<DBNotification variant="standalone" semantic="informational">
Only use multi-line text in exceptional cases, such as when the text is extended due to translation.
</DBNotification>


To enable automatic line breaks in a full-width button, select the "Text" layer inside the button and set the width resizing behavior from "Hug" to "Fill".
