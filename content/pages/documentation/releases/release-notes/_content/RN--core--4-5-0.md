### Features Development

- New `propOverrides` property for components to avoid duplicate `id` attributes in Angular and Web Components.
- New Stylelint rule `db-ux/use-sizing` in the `@db-ux/core-stylelint` package, which checks for direct usage of `height`, `width`, `block-size` etc.
