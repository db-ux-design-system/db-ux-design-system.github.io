### Design

**chore(components): component level versioning**

- Starting with this release, every component will be tagged with the version number of its latest update in the background — invisible to users, with no impact on existing designs. This information serves as the foundation for the migration plugin, which will support assisted migration for upcoming major releases.
- Recommendation: Keep your component instances up to date — only up-to-date instances can be supported during migration for the major release.
