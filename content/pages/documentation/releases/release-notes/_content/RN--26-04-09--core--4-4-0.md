### Development

- feat(select): add `showEmptyOption` property and hide empty option for required selects by default
  - For required selects with a placeholder, the empty first option is automatically hidden after the user's first selection. This behavior can be overridden via the new `showEmptyOption` prop.
- feat(notification): add default ARIA role attributes for accessibility
  - New `role` property. If neither `role` nor `ariaLive` is set, the role is automatically assigned based on the semantic — for a better screen reader experience.
- refactor: extending `peerDependency` range to Stylelint 17
