### Refinements

The following refinements require special attention. Existing instances must be reviewed after updating.

#### Scrollbar

- **Resizer helper component refactored:** The Scrollbar now uses the Resizer as a dedicated helper component. Components using the Scrollbar must be reviewed to verify that Handle Size and Free Position values are still correct after the update.

<YourAction
	actions={[
		'Review all components that use the Scrollbar helper (e.g. Custom Select Dropdown, Shell, vertical navigations).',
		'Verify that the "Thumb Resizer" gap value still produces the correct handle size.',
		'Verify that the "Spacer Resizer" gap value still produces the correct free position.',
	]}
/>

</DBSection>

#### Header & Navigation

- **Replaced by ControlPanel + Shell:** The Header and Navigation components are deprecated. Use ControlPanel and Shell instead. Header and Navigation will be fully removed with v7.0.

In Figma, affected deprecated components are marked with **"❌ ... (Deprecated)"** in the name.

<YourAction
	actions={[
		'Identify all instances of Header and Navigation in your designs.',
		'Replace Header with the appropriate Shell + ControlPanel combination (Desktop or Mobile).',
		'Replace Navigation with ControlPanel Navigation or Shell SubNavigation.',
		'Refer to the Shell Desktop and Shell Mobile documentation for configuration guidance.',
	]}
/>

</DBSection>
