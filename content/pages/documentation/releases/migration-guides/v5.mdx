### Refactoring

The following refactoring requires special attention. Existing instances must be reviewed after updating.

#### Drawer

- **Standardized layout structure:** The Drawer component has been refactored with unified subcomponents (DrawerHeader + DrawerFooter) and standard container sizes (small, medium, large, full, custom). Previous Drawer components are replaced by the new version.

<YourAction
	actions={[
		'Review all deployed Drawer instances in your designs.',
		'Replace deprecated Drawer variants with the new standardized structure.',
		'Ensure DrawerHeader and DrawerFooter subcomponents are used correctly.',
		'Verify container size (small, medium, large, full, custom) matches your layout needs.',
	]}
/>
</DBSection>

<DBSection spacing="small" width="auto">

### Refinements

The following refinements require special attention. Existing instances must be reviewed after updating.

#### Scrollbar

- **Resizer helper component refactored:** The Scrollbar now uses the Resizer as a dedicated helper component. Components using the Scrollbar must be reviewed to verify that Handle Size and Free Position values are still correct after the update.

<YourAction
	actions={[
		'Review all components that use the Scrollbar helper (e.g. Custom Select Dropdown, Shell, vertical navigations).',
		'Verify that the "Thumb Resizer" gap value still produces the correct handle size.',
		'Verify that the "Spacer Resizer" gap value still produces the correct free position.',
	]}
/>

#### Card & Popover

- **Default slot visibility aligned with system standard:** The "Start Slot" and "End Slot" now default to hidden (Show Slot = false) when empty, matching the standard behavior across all components. If your designs rely on visible Start or End slots with content, they must be re-enabled after the update.

<YourAction
	actions={[
		'Review all Card and Popover instances that use Start Slot or End Slot content.',
		'Re-enable slot visibility (Show Slot = true) where Start or End content is needed.',
	]}
/>
</DBSection>

#### Header & Navigation

- **Replaced by ControlPanel + Shell:** The Header and Navigation components are deprecated. Use ControlPanel and Shell instead. Header and Navigation will be fully removed with v7.0.

In Figma, affected deprecated components are marked with **"❌ ... (Deprecated)"** in the name.

<YourAction
	actions={[
		'Identify all instances of Header and Navigation in your designs.',
		'Replace Header with the appropriate Shell + ControlPanel combination (Desktop or Mobile).',
		'Replace Navigation with ControlPanel Navigation or Shell SubNavigation.',
		'Refer to the Shell Desktop and Shell Mobile documentation for configuration guidance.',
	]}
/>
