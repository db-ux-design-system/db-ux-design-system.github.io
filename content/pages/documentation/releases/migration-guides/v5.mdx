### Refactoring

The following refactoring requires special attention. Existing instances must be reviewed after updating.

#### Drawer

- **Standardized layout structure:** The Drawer component has been refactored with unified subcomponents (DrawerHeader + DrawerFooter) and standard container sizes (small, medium, large, full, custom). Previous Drawer components are replaced by the new version.

<YourAction
	actions={[
		'Review all deployed Drawer instances in your designs.',
		'Replace deprecated Drawer variants with the new standardized structure.',
		'Ensure DrawerHeader and DrawerFooter subcomponents are used correctly.',
		'Verify container size (small, medium, large, full, custom) matches your layout needs.',
	]}
/>

#### Accordion

- **Children Slot replaces 🛟 Accordion List helper component:** The ❌🛟 Accordion List helpercomponent has been marked as deprecated and is replaced by a Children Slot. Accordion Items are now placed directly inside the Accordion via the slot.

<YourAction
	actions={[
		'Review all Accordion instances in your designs.',
		'Replace ❌🛟 Accordion List usage with the new Children Slot.',
		'Place Accordion Items directly inside the Accordion component.',
	]}
/>
</DBSection>

<DBSection spacing="small" width="auto">

### Refinements

The following refinements require special attention. Existing instances must be reviewed after updating.



#### Card & Popover

- **Default slot visibility aligned with system standard:** The "Start Slot" and "End Slot" now default to hidden (Show Slot = false) when empty, matching the standard behavior across all components. If your designs rely on visible Start or End slots with content, they must be re-enabled after the update.

<YourAction
	actions={[
		'Review all Card and Popover instances that use Start Slot or End Slot content.',
		'Re-enable slot visibility (Show Slot = true) where Start or End content is needed.',
	]}
/>
</DBSection>

<DBSection spacing="small" width="auto">

### Removed

The following components and subcomponents were deprecated in v4.6 and are now fully removed. If you have not migrated yet, immediate action is required.

#### Card, Popover, Drawer & Section

- **Deprecated component variants removed:** The old component variants (using the slot helper component) have been removed. Use the current versions with native Figma slots instead.

<YourAction
	actions={[
		'Identify any remaining instances of ❌ Card, ❌ Popover, ❌ Drawer, or ❌ Section (Deprecated) in your designs.',
		'Replace them with the current component versions using native Figma slots.',
	]}
/>

#### Accordion Item & Tag – Slot Helper Components

- **Deprecated slot helper subcomponents removed:** The dedicated slot helper components for Accordion Item and Tag have been removed. Use the native Figma slot mechanism directly.

<YourAction
	actions={[
		'Identify any remaining instances using the deprecated slot helper in Accordion Item or Tag.',
		'Replace them with the current component versions using native Figma slots.',
	]}
/>

</DBSection>
