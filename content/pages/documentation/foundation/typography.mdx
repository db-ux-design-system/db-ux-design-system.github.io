## Guidelines

###### Code Mapping

Our typography has a simple and effective code mapping system that links the most common HTML tags (h1, h2, h3, h4, h5, h6, p, small) directly to our typography tokens 3xs to 3xl.
If this is not sufficient for a particular use case, specific tokens can be used instead.
