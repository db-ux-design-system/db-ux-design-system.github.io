## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

###### Checkbox vs Radio

Use checkboxes when users can select any number of options (including none). If the user must pick exactly one option, use Radio instead.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"

		description="Use checkboxes where multiple selections are allowed."
	/>
	<GuidelineExample

	/>

###### Labels

Labels must be clear and concise. Prefer positive phrasing to reduce cognitive load and avoid double negatives.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/3sTqkyZLQuIgT27PyPmIbz/Doku--Checkbox-%E2%9C%85?node-id=3-1110&embed-host=share"
		description="Use clear and concise labels."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/3sTqkyZLQuIgT27PyPmIbz/Doku--Checkbox-%E2%9C%85?node-id=3-1112&embed-host=share"
		description="Avoid double negatives that can confuse when checked."
	/>
</div>

###### Grouping

In checkbox groups, keep options parallel in wording and length. If a group needs context, add a group label or short description above it.

<div className="guideline-examples">
	<GuidelineExample
