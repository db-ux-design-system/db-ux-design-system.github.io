Custom Select is an advanced dropdown for choosing one or multiple predefined options with support for search, grouping, and showing selected values. Use it when a basic select is not sufficient due to option volume, selection complexity, or the need for better scanning and management of chosen items.

- Select multiple items from a longer list with optional "Select all"
- Search and filter within options to quickly find values
- Manage selected options clearly (e.g., chips/tags or selected list)

## Usage Hints

###### Custom Select vs Select

Prefer select for simple single-choice lists. Use custom select when you need multi-select, search, grouping, or a clearer representation of multiple selected values.

###### Single- or multiselect

Decide early whether the control is single-select or multi-select. If multi-select is allowed, make it obvious and keep selection behavior consistent.

###### Selected options

Ensure selected options are always easy to review without reopening the list. Use "Tags" with remove buttons for frequent changes, "Amount" in tight spaces (e.g., tables), or "Text" for single selections. Always matching the presentation to the context.

###### Searching & Grouping

Enable search when the list is long or users know what they want. Use grouping only when it reflects meaningful categories; keep group labels clear and consistent.
