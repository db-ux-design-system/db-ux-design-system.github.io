Section separates and structures content areas on a page by applying consistent spacing and optional width constraints. It helps create clear visual grouping and rhythm without manually managing margins for each block.

- Group related content into a distinct page area (e.g., "Overview", "Details", "Next steps")
- Create consistent vertical spacing between major content blocks
- Constrain content width for readability while keeping full-bleed backgrounds possible

## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

###### Structuring

Use Section for high-level grouping of content blocks. It should define consistent spacing between major areas, not micro-spacing between individual elements.

###### Spacing and Width

Try to choose the spacing and width options as consistent as possible and suitable for the respective context.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/YER9qlSt9hGtlnFAqe2s7s/Doku--Section-%E2%80%BC%EF%B8%8F?node-id=3-1110&embed-host=share"

	/>

###### Accessibility

Section is a layout tool, not a semantic substitute. Ensure content inside is properly structured with headings and landmarks when appropriate (e.g., meaningful heading levels, ARIA landmarks only when needed).
