Stack arranges child elements in a vertical or horizontal layout with consistent spacing. Use it to create structured layouts without writing custom CSS.

- Arrange form elements, buttons, or content blocks with consistent spacing
- Create vertical or horizontal layouts without custom CSS
- Build responsive layouts that adapt to different screen sizes

## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

###### Gap

Use the 'gap' property to provide consistent spacing between items in a Stack, improving readability and visual structure.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=1-1132&embed-host=share"
		description="Use a defined gap to evenly space items and create a clean, organized layout."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=1-1133&embed-host=share"
		description="Don't manually add margins to individual items instead of using gap. It can lead to inconsistent spacing and harder maintenance."
	/>
</div>

###### Direction and alignment

Choose the appropriate direction (column or row) and alignment based on your layout and content needs.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=3-1110&embed-host=share"
		description="Use horizontal (row) direction to arrange related actions making them quickly accessible and visually grouped."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=3-1112&embed-host=share"
		description="Don't use center alignment in columns if the content has different lengths and is intended to be quickly scanned."
	/>
</div>

###### Wrapping

Enable wrap for stacks that should adapt to smaller screens.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=3-1113&embed-host=share"
		description="Use wrap when you have many items or expect it to be used on different screen sizes."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=3-1114&embed-host=share"
		description="Don't use “No Wrap” when there's a risk of overflowing and the content should be easily accessible."
	/>
</div>
