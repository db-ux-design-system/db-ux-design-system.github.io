## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

###### Consistent spacing

Use Stack to maintain consistent spacing between elements instead of adding margins manually to each child.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/uBCf96A16t1871VinjYHHV/Doku--Stack?node-id=1-1132&embed-host=share"
		description="Use Stack with a defined gap to create consistent spacing between elements."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/[designId]?node-id=[nodeId]&embed-host=share"

	/>
</div>

###### Direction and alignment

Choose the appropriate direction (column or row) and alignment based on your layout needs.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/[designId]?node-id=[nodeId]&embed-host=share"
		description="Use column direction for vertical lists and row direction for horizontal button groups."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/[designId]?node-id=[nodeId]&embed-host=share"
		description="Don't use row direction for long lists of content that should stack vertically."
	/>
</div>

###### Divider variant

Use the divider variant to visually separate stacked elements when needed.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/[designId]?node-id=[nodeId]&embed-host=share"

	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/[designId]?node-id=[nodeId]&embed-host=share"
		description="Don't use divider variant without adding DBDivider components."
	/>
</div>

###### Responsive wrapping

Enable wrap for horizontal stacks that should adapt to smaller screens.

	<GuidelineExample
