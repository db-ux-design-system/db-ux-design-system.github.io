Navigation Item is a subcomponent used within Navigation to represent individual destinations. It supports icons, active states, disabled states, and nested sub-navigation for hierarchical structures.

- Always wrap content in an anchor element for proper navigation semantics
- Use the active property or aria-current="page" to indicate the current location
- Add icons to improve scannability and recognition
