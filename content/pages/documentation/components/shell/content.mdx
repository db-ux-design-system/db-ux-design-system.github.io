## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

### Variant (Auto vs. Fixed)

Use Auto when the entire content area (Start Slot, Children and End Slot) should scroll together. Use Fixed when the Start Slot or End Slot must remain persistently visible while only the Children slot scrolls.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-3&embed-host=share"
		description="Use Fixed when docked notifications or a persistent Footer must stay visible while scrolling."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-4&embed-host=share"
		description="Do not overfill the Start Slot or End Slot in Fixed. Large slots shrink the Children scroll area to an unusable size."
	/>
</div>

### Start Slot

The Start Slot is intended for persistent, system-level UI elements such as docked notifications that must remain visible above the main content.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-6&embed-host=share"
		description="Place docked notifications or system banners in the Start Slot."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-7&embed-host=share"
		description="Do not place regular page content in the Start Slot. It is reserved for persistent UI elements."
	/>
</div>

### End Slot

The End Slot stays at the bottom of the content area. A Footer with legal links can scroll with the content in Auto. Important elements like a primary CTA must use Fixed, otherwise they scroll out of view and become unreachable.

<div className="guideline-examples">
	<GuidelineExample

		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-9&embed-host=share"

	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/RcpbbqfJwjNRlOwPhAX1im/Doku--Shell?node-id=4079-10&embed-host=share"
		description="Do not place important actions in an Auto End Slot. They scroll out of view before users reach them."
	/>
</div>
