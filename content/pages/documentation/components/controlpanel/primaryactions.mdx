## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

### Count

Keep PrimaryActions minimal. Only place 1–3 high-frequency actions that users need on every page.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-2&embed-host=share"
		description="Use 1–2 focused global actions like Search that users need on every page."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-4&embed-host=share"
		description="Do not overload PrimaryActions with too many buttons. Too many actions dilute priority and clutter the ControlPanel."
	/>
</div>

### Action Type

PrimaryActions should only contain global actions that apply to every page — not content-specific actions that belong on the page itself.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-6&embed-host=share"
		description="Use global actions like Search that are relevant regardless of which page the user is on."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-8&embed-host=share"
		description="Do not place content-specific actions like Save in PrimaryActions. These belong on the page level, not in the global navigation bar."
	/>
</div>

### FlatIcon Limitation

The FlatIcon ControlPanel does not support PrimaryActions. If your application requires always-visible primary actions, use Top or Bottom position instead.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-10&embed-host=share"
		description="Use Top or Bottom ControlPanel when primary actions need to be directly accessible in the navigation bar."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4338-12&embed-host=share"
		description="Do not place actions like Search as FlatIcon NavigationItems. FlatIcon is reserved for navigation destinations — use Top or Bottom if actions are needed."
	/>
</div>
