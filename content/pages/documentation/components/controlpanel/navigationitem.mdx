- Popover Item: Flat navigation link. Popover Group opens a flyout (Desktop) or navigates to a sublevel (Mobile)
- Tree Item: Navigation link within a tree structure. Icons are overridable for Items only — not for Groups
- FlatIcon Item: Icon-based navigation destination. Tooltip text equals the Text Screenreader value
- All variants support interaction states: Enabled, Hovered, Pressed, Focused, Disabled
- All variants support an Active state indicating the current page/section

### Slots

Slot availability varies by variant. The FloatingContainer (for tooltips, popovers, badges) is available on all NavigationItem variants. The End Slot for inline content is only available on vertical Popover and Tree Items. FlatIcon Items and horizontal Popover Items have no content slots.

### Tree Icons

Tree NavigationItems allow overriding the default tree-line icon with a custom icon. This applies to Items only — Groups always keep the expand/collapse chevron.

### FlatIcon Tooltip

FlatIcon NavigationItems display a tooltip on hover that matches the Text Screenreader value. This ensures consistency between the visual tooltip and what assistive technologies announce.
