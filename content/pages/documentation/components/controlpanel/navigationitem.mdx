## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

### Icon Consistency per Level

Tree NavigationItems allow overriding the default tree-line marker with a custom icon (Items only – Groups always keep the expand/collapse chevron). Keep icons consistent within a level: either all sibling items on a level get a custom icon or none. Levels may differ from each other.

### Slots

Slot availability varies by variant. The FloatingContainer (for tooltips, popovers, badges) is available on all NavigationItem variants. The End Slot for inline content is only available on vertical Popover and Tree Items. FlatIcon Items and horizontal Popover Items have no content slots.

## Accessibility

### FlatIcon Tooltip

FlatIcon NavigationItems display a tooltip on hover that matches the Text Screenreader value. This ensures consistency between the visual tooltip and what assistive technologies announce. Provide a clear, descriptive text for each FlatIcon item – it serves as both tooltip and accessible label. Do not leave it empty or use a different text than the tooltip.
