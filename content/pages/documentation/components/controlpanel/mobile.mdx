## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

### Density

The mobile ControlPanel only supports Regular density. Touch targets need sufficient size for reliable finger interaction on mobile devices.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4172-2&embed-host=share"
		description="Use Regular density for the mobile ControlPanel to ensure comfortable touch targets."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4172-4&embed-host=share"
		description="Do not apply Functional or Expressive density to the mobile ControlPanel. Only Regular is supported on mobile."
	/>
</div>

### FlatIcon Usage

FlatIcon (tab bar) works best for apps with a small set of equally important top-level destinations. It provides instant access without opening a Drawer but has limited capacity.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4172-6&embed-host=share"
		description="Use FlatIcon when the application has 2–6 equally weighted top-level sections that users switch between frequently."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4172-8&embed-host=share"
		description="Do not use FlatIcon with labels that are too long. Labels get truncated and users cannot distinguish destinations without reading the full text."
	/>
</div>

## Examples

<CollapsibleWrapper collapsedHeight="320px">

### Position

Select the ControlPanel position based on the app type and primary interaction pattern. Top is the default for content-focused apps. Bottom works for app-like experiences where navigation actions need to be within thumb reach.

<div className="guideline-examples">
	<GuidelineExample
		figmaUrl="https://embed.figma.com/design/pElrqVUyojrzYzSagyJPS6/Doku--ControlPanel?node-id=4172-10&embed-host=share"
		description="Top position for content-focused applications where navigation is secondary to reading or consuming content."
	/>
