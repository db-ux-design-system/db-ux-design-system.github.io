## FAQ

<DBAccordion>
	<DBAccordionItem headlinePlain="When should I use a Link instead of a Button?">
		Use a Link for navigation. Use a Button for actions that trigger changes, submit data, or
		manipulate content.
	</DBAccordionItem>
	<DBAccordionItem headlinePlain="Can a Link include an icon?">
		Yes, links may include an icon to indicate direction or external navigation, as long as the
		label remains clear.
	</DBAccordionItem>
	<DBAccordionItem headlinePlain="Are links allowed inside text paragraphs?">
		Yes, but use dedicated text-link styles rather than the standalone Link component.
	</DBAccordionItem>
</DBAccordion>
