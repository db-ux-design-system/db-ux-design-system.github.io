The link component is used for navigation to other pages, sections, or external destinations and visually indicates a navigational affordance within the interface.

- Navigate to another page within the application
- Navigate to an external website or document
- Jump to a specific section within the same page

## Usage Hints

###### Purpose

Use the link component exclusively when the primary purpose is navigation to a different location or resource.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/3HjaNLCvfdsdoRZKbFnV3s/Doku--Link?node-id=1-1132&embed-host=share"
		description="Use a link to navigate to another page or section."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/3HjaNLCvfdsdoRZKbFnV3s/Doku--Link?node-id=1-1133&embed-host=share"
		description="Don't use a link to trigger an action or change data."
	/>
</div>

###### Inline

For links within running text, use text-link styles instead of the standalone link component.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/3HjaNLCvfdsdoRZKbFnV3s/Doku--Link?node-id=3-1110&embed-host=share"
		description="Use text links inside paragraphs for inline navigation."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/3HjaNLCvfdsdoRZKbFnV3s/Doku--Link?node-id=3-1112&embed-host=share"
		description="Don't place the link component inside continuous body text."
	/>
</div>

###### Emphasis

Use the brand variant sparingly to highlight particularly important navigational links.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"

	/>

		description="Don't overuse emphasized link styles for all links on a page."
	/>
</div>
