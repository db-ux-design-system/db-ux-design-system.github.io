Divider is a thin horizontal or vertical line that separates UI elements into logical sections and improves layout clarity. It should be used sparingly and together with spacing and hierarchy to avoid visual clutter.

- Separate content blocks within a page or card (horizontal divider)
- Split columns or side-by-side regions in a layout (vertical divider)
- Add a subtle boundary between groups of related items (e.g., list sections)

## Guidelines

<CollapsibleWrapper collapsedHeight="320px">

### Emphasis

Use weak for subtle separation. Use strong only when you need a more pronounced boundary between major sections and the layout would be unclear with spacing alone.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/gdUAKTrAb1eXjhYDGpzUUZ/Doku--Divider?node-id=1-1132&embed-host=share"
		description="Use weak emphasis for subtle separation without disrupting the user flow."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/gdUAKTrAb1eXjhYDGpzUUZ/Doku--Divider?node-id=1-1133&embed-host=share"
		description="Avoid strong dividers everywhere or to compensate for missing layout structure."
	/>
</div>

### Usage

Dividers support grouping, but spacing and headings create the primary structure. Add a divider only when spacing alone does not provide enough separation or scanning support.

<div className="guideline-examples">
	<GuidelineExample
		variant="do"
		figmaUrl="https://embed.figma.com/design/gdUAKTrAb1eXjhYDGpzUUZ/Doku--Divider?node-id=3-1110&embed-host=share"
		description="Use consistent spacing tokens and add a divider only where it improves comprehension."
	/>
	<GuidelineExample
		variant="dont"
		figmaUrl="https://embed.figma.com/design/gdUAKTrAb1eXjhYDGpzUUZ/Doku--Divider?node-id=3-1112&embed-host=share"
		description="Don't add dividers between every element, creating visual noise and reducing hierarchy."
	/>
</div>

### Alignment

Align dividers to the content container and grid. Avoid partial or "floating" dividers that don't match content width/height and create accidental alignment breaks.
