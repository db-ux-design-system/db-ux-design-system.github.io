Active beschreibt ein Element, das aktuell in der Interface markiert, ausgewählt oder hervorgehoben ist. Dieser Zustand wird typischerweise erzeugt, indem dem Element eine höhere visuelle Hierarchie als seinem Standard-Enabled-Zustand gegeben wird. Dieses Pattern schreibt keinen einzelnen festen Stil vor, sondern beschreibt eine Reihe von Methoden, die einzeln oder in Kombination verwendet werden können.

Je nach Kontext, Hierarchie und Interface-Dichte kann der Active State über den Container (Fläche, Farbe, Rahmen) oder über den Inhalt (Typografie, Icon) ausgedrückt werden. Die folgenden Varianten zeigen gängige Ansätze, die konsistent kombiniert und über verschiedene Komponenten hinweg angepasst werden können.

## Container

### Fläche

Die erste Möglichkeit besteht darin, die visuelle Betonung des Containers zu erhöhen. Dies kann durch stärkeren Kontrast oder sogar invertierte Vorder- und Hintergrundfarben erreicht werden, sodass der Active State sofort hervorsticht.

### Farbe

Eine weitere Option ist die Anpassung der Container-Farbe. Ein verschobener oder getönter Hintergrund unterscheidet den Active State von inaktiven Elementen, ohne das Gesamtlayout zu dominieren. Dies kann einfach durch Änderung des data-color angewendet werden.

</div>

### Rahmen

Der Active State kann auch durch einen prominenteren Rahmen dargestellt werden, beispielsweise mit erhöhter Strichstärke oder Farbwechsel. Die veränderte Kontur macht das aktive Element klar erkennbar, selbst wenn die Flächen visuell zurückhaltend bleiben.

## Inhalt

### Typografie

Anstatt den Container zu verändern, kann der Active State auch durch Typografie hervorgehoben werden. Ein höheres Schriftgewicht oder andere typografische Akzente sorgen dafür, dass das Label des aktiven Elements im direkten Vergleich deutlich dominanter erscheint.

### Inhalt

Der Active State kann auch durch Änderung oder Erweiterung des Inhalts ausgedrückt werden. Ein zusätzliches Icon, wie ein Häkchen, oder ein verändertes Label macht die Auswahl semantisch klar und verstärkt die visuelle Betonung.
